# API de Turnos Hospitalarios

API REST desarrollada con FastAPI para la gestión de pacientes, profesionales y turnos médicos.

## Funcionalidades
- Alta de pacientes
- Alta de profesionales
- Creación de turnos
- Validación de solapamiento de turnos
- Cancelación y atención de turnos

## Tecnologías
- Python
- FastAPI
- Pydantic
